Major Defect

SECTION MARK

Weaving Defect

Description

Visible lines or marks running across the width of fabric, typically appearing as distinct bands or streaks that differ in appearance from the rest of the fabric.

  • Appears as horizontal bands
  • May show color variations
  • Creates uneven fabric appearance

Causes

  • Loom stoppages during weaving
  • Tension variations in warp yarns
  • Different yarn lots with varying properties
  • Inconsistent beat-up pressure
  • Reed setting variations
  • Temperature/humidity changes

Remedies

  • Maintain consistent loom settings
  • Use uniform yarn lots
  • Regular tension monitoring
  • Proper conditioning of warp
  • Minimize loom stoppages
  • Train operators on consistent practices
Minor Defect

WEFT CHANGE

Weaving Defect

Description

Visible line or mark in the fabric where one weft yarn package was changed to another during weaving process.

  • Appears as a horizontal line
  • May show slight color variation
  • Usually minor visual impact

Causes

  • Different yarn lots used
  • Slight color variations between packages
  • Tension differences in new package
  • Poor knotting at changeover
  • Yarn count variations

Remedies

  • Use yarns from same dye lot
  • Proper tension adjustment
  • Skilled package changing
  • Quality matching of yarn packages
  • Minimize visible changeover points
